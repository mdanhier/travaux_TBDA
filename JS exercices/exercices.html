Exercice 1

  1. Créer un script qui demande successivement à l'utilisateur son nom puis son prénom. La page affichera les informations sur l'utilisateur via une "alert()".

  2. Ecrire un programme qui calcule le produit de deux nombres entrés par l'utilisateur et affiche le résultat.

  3. Ecrire un programme qui convertit des degrés Celsius en degrés Fahrenheit en utilisant la formule suivante : F = (C*9/5) + 32. Le programme demandera la température en degrés Celsius à l'utilisateur et affichera la température convertie en degrés Fahrenheit.

Exercice 2

Soit les variables suivantes :

  • a qui contient la chaîne de caractères 100
  • b = 100
  • c qui contient la valeur 1,00
  • d booléen qui vaut vrai

A réaliser :

  1. Affichez "Ceci est une chaîne de caractères :" et concaténez cette chaîne avec la variable a pour afficher "Ceci est une chaîne de caractères : 100".
  2. Appliquez à b l'opérateur de décrémentation
  3. Ajoutez à c la valeur de a
  4. Inversez la valeur de d

Exercice 3

  1. Ecrivez un programme qui demande un nombre à l'utilisateur puis qui teste si ce nombre est pair. Le programme doit afficher le résultat nombre pair ou nombre impair. Vous devez utiliser l'opérateur modulo % qui donne le reste d'une division. a%2 donne le reste de la division de a par 2, si ce reste est égal à zéro, a est divisible par 2.

  2. Ecrivez un programme qui demande l'année de naissance à l'utilisateur. En réponse votre programme doit afficher l'âge de l'utilisateur et indiquer si l'utilisateur est majeur ou mineur.

  3. Faire la saisie de 2 nombres entiers, puis la saisie d'un opérateur +, -, * ou /.

    Si l'utilisateur entre un opérateur erroné, le programme affichera un message d'erreur.

    Dans le cas contraire, le programme effectuera l'opération demandée (en prévoyant le cas d'erreur division par 0), puis affichera le résultat.
